📚 Cómo funciona el sistema

🪪 CÓMO SE CALCULA LA LETRA DEL DNI
letra = "TRWAGMYFPDXBNJZSQVHLCKE"[numero % 23]
Ejemplo: 12345678 % 23 = 14 → letra "Z" → 12345678Z
Rango total: 00000001A → 99999999Z = 99.999.999 DNIs posibles
🚗 FORMATO DE MATRÍCULA ESPAÑOLA (desde 2000)
NNNN LLL (4 dígitos + espacio + 3 letras)
Letras válidas (20): B C D F G H J K L M N P R S T V W X Y Z
Se excluyen: A E I O U (vocales) + Ñ Q U V (ambigüedad visual/fonética)
Primera matrícula: 0000 BBB → Última teórica: 9999 ZZZ
Total combinaciones: 10.000 × 20³ = 80.000.000 matrículas
🔧 CÓMO FUNCIONA ESTE BUSCADOR
• Si el DNI o matrícula está en la BD → devuelve los datos guardados (badge azul EN BD)
• Si NO está en la BD → genera datos ficticios al vuelo usando el número como semilla aleatoria (GENERADO)
• Los datos generados son reproducibles: el mismo número siempre produce el mismo resultado
• Las alertas policiales se calculan determinísticamente: numero % 100 < 7 → Busca y Captura, etc.
